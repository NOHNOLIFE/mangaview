## manual
### mouse `click`  `drag`

### keyboard:
* scroll down: `space`\ `↓`
* page down: `F`\ `→`\ `enter`
* 
* scroll up: `shift + space`\ `↑`
* page up:`D`\ `←`
* 
* next book: `page-down`
* previous book:  `page-up`
